# Resource

## About

Every class.Player has own resource class, where are starting resources

## Attributes

- **food**
- **wood**
- **gold**
- **stone**
- **ore** not-used in AOE2

## Methods

- **toJSON()**
- **setAll(value)** for all resources set a value
- **getAll()** get resource as tuple(food, wood, gold, stone, ore)
